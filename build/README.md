# Build assets / 构建资源

This directory contains source assets and platform metadata used by Wails. Generated applications and release archives are written to `build/bin/` and are intentionally excluded from Git.

本目录包含 Wails 构建所需的源素材和平台元数据。生成的应用与发布压缩包位于 `build/bin/`，并被 Git 忽略。

- `appicon.svg` / `appicon.png`: application icon sources / 应用图标源文件
- `darwin/`: macOS plist metadata / macOS plist 元数据
- `windows/`: reserved Windows metadata and installer templates / 预留的 Windows 元数据与安装模板
