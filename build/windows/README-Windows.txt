Codex Canvas 1.1.0 - Windows x64 Preview
================================================

ENGLISH

Requirements:
- Windows 10/11 x64
- Official ChatGPT desktop app installed from Microsoft Store
- Microsoft Edge WebView2 Runtime (normally included with Windows 10/11)

Run Codex-Canvas.exe directly. Windows may display a SmartScreen warning because
this community preview is not Authenticode-signed. Only continue if you downloaded
the file from the official GitHub Release for this repository.

The Windows adapter discovers the Microsoft Store package dynamically and does not
modify official application files. This first Windows build has been cross-compiled
and statically verified, but has not yet been tested on a Windows machine. Please
report compatibility results through GitHub Issues without including private data.

中文

运行要求：
- Windows 10/11 x64
- 已从 Microsoft Store 安装官方 ChatGPT 桌面应用
- Microsoft Edge WebView2 Runtime（Windows 10/11 通常已自带）

直接运行 Codex-Canvas.exe。由于社区预览版尚未进行 Authenticode 签名，
Windows 可能显示 SmartScreen 提示。仅当文件来自本仓库官方 GitHub Release 时继续。

Windows 适配会动态发现 Microsoft Store 应用包，不修改官方应用文件。此首个 Windows
版本已完成交叉编译和静态验证，但尚未在 Windows 真机测试。欢迎通过 GitHub Issues
反馈兼容结果，请勿包含隐私数据。
