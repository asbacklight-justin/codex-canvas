# 隐私说明

**简体中文** | [English](PRIVACY.md)

Codex Canvas 1.1.0 按本地运行方式设计。

- 项目不运营后端服务。
- 不收集分析、遥测、崩溃报告、账户数据、提示词或 Codex 对话。
- 所选图片只在本机读取，并仅复制到 `~/Library/Application Support/Codex Canvas`。
- 应用主题时只连接本机 Codex 进程在 `127.0.0.1` 暴露的随机端点。
- 不下载或执行远程主题包。

操作系统、Codex Desktop、GitHub 和安装包下载渠道有各自的隐私政策，不受本项目控制。
