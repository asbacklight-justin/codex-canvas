# Privacy

[简体中文](PRIVACY.zh-CN.md) | **English**

Codex Canvas 1.1.0 is designed to work locally.

- It does not operate a backend service.
- It does not collect analytics, telemetry, crash reports, account data, prompts, or Codex conversations.
- Selected images are read locally and copied only to `~/Library/Application Support/Codex Canvas`.
- Theme application connects only to a random endpoint on `127.0.0.1` exposed by the locally running Codex process.
- The app does not download or execute remote theme packages.

The operating system, Codex Desktop, GitHub, and any download channel have their own privacy practices. They are outside this project's control.
